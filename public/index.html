Niveles Técnicos

Es necesario realizar una clase de prueba para la colocación del grupo y/o nivel.

  1. Matronatación

    Bebés

    11 meses a 2.5 años

    Requisitos especiales

    • Uso de Pañal Nadador
    • Acompañado de un familiar directo
    • Revisar que a la hora de clase NO sea hora de siesta
    • Indispensable dejar pasar al menos una hora después de comer

    Matronatación

    Para alumnos de 11 meses a 2.5 años
